什么是Github

Github是全球最大的社交编程及代码托管网站。
Github可以托管各种git库，并提供一个web界面（用户名.github.io/仓库名）

使用Github目的

GitHub 是一个面向开源及私有软件项目的托管平台，因为只支持 Git 作为唯一的版本库格式进行托管，故名 GitHub。
Git可以有效、高速的处理从很小到非常大的项目版本管理，我们使用GitHub的目的是借助GitHub托管项目的代码。
